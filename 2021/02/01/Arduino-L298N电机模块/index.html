L298N模块

模块简介

L298N 是一种双H桥电机驱动芯片.

L298N电机驱动模块是一款接受高电压的电机驱动器，可驱动直流电机和步进电机.

接口

VCC:模块供电需12V.

GND:连接电源负极.

5V:模块对外输出5V，当板载5V使能端断开时为模块输入.

ENA1:控制电机输出A.

IN1 IN2:逻辑输入.

OUT1:电机输出.

逻辑输入

ENA IN1 IN2 直流电机状态
0 0 0 停止
1 0 0 制动
1 0 1 正转
1 1 0 反转
1 1 1 制动

接线

博主使用Arduino UNO板子.

闭合板载5V使能和通道A使能（使用PWM调速时A使能接UNO 5）.

VCC+GND–12V外接电源

5V–UNO Vin

OUT1–电机正负极

IN1–UNO 2

IN2–UNO 3
